## :ledger: Overview
Coffee Talks is a project designed to promote German language learning and cultural exchange within an international company based in Germany. This initiative aims to help employees improve their German language skills, gain cultural understanding, and enhance collaboration across diverse teams. By fostering a welcoming and inclusive environment, we encourage employees to connect, learn, and grow together.
